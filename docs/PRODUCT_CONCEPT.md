## 1. Executive Summary

### Что это?

**Jezkazgan Connect** (Афиша Жезказган) — это городская digital-платформа, объединяющая жителей города Жезказган (Казахстан) с местными бизнесами, событиями и сообществами. Платформа решает проблему информационной разрозненности в небольшом городе, создавая единую экосистему для:

- Поиска и продвижения городских мероприятий
- Каталогизации местного бизнеса
- Распространения скидок и акций
- Создания городских сообществ по интересам
- Организации B2B коллабораций

### Ключевые цифры рынка

| Показатель | Значение |
|------------|----------|
| Население Жезказгана | ~90,000 человек |
| Активные интернет-пользователи (оценка) | ~60,000 (67%) |
| Количество МСБ в городе (оценка) | ~2,000-3,000 |
| Потенциальная аудитория платформы | 40,000-50,000 пользователей |

### Текущий статус проекта

**Стадия:** MVP/Demo (35-40% готовности)
**Frontend:** Полностью разработан (React + TypeScript)
**Backend:** Не реализован
**Данные:** Mock данные для демонстрации

## 2. Vision & Mission

### Vision (Видение)

> Стать центральной digital-платформой городской жизни Жезказгана, где каждый житель может узнать о событиях, поддержать местный бизнес и найти единомышленников.

### Mission (Миссия)

> Объединить жителей Жезказгана с местными бизнесами и событиями через удобную цифровую платформу, способствуя развитию городской экономики и укреплению локального сообщества.

### Принципы продукта

1. **Local First** — фокус на локальном контенте и бизнесе
2. **Простота** — интуитивный интерфейс для всех возрастов
3. **Доверие** — верификация бизнесов и модерация контента
4. **Взаимовыгода** — ценность для жителей и бизнеса одновременно

## 3. Анализ рынка

### 3.1 Контекст города Жезказган

**Жезказган** — город в Карагандинской области Казахстана с населением около 90,000 человек. Исторически это моногород, развивавшийся вокруг медной промышленности.

**Характеристики города:**
- Географическая изолированность (300+ км до Караганды)
- Молодое население (средний возраст ~35 лет)
- Высокий уровень проникновения интернета (~67%)
- Активное использование мессенджеров (Telegram, WhatsApp)
- Развивающийся сектор МСБ

### 3.2 Проблемы, которые решает продукт

| Проблема | Для кого | Как решаем |
|----------|----------|------------|
| Отсутствие единой афиши событий | Жители | Централизованный каталог событий с фильтрами |
| Сложность продвижения для МСБ | Бизнес | Бесплатные и платные инструменты продвижения |
| Разрозненная информация об акциях | Жители | Агрегатор скидок с поиском |
| Отсутствие нетворкинга | Бизнес | Раздел коллабораций |
| Нет community-платформы | Все | Сообщества по интересам |

### 3.3 Рыночные тренды

1. **Локализация digital-сервисов** — рост спроса на гиперлокальные платформы
2. **Super App тренд** — пользователи предпочитают одно приложение для множества задач
3. **Поддержка МСБ** — государственная политика поддержки малого бизнеса в Казахстане
4. **Digital-трансформация регионов** — программы "Цифровой Казахстан"

## 4. Целевая аудитория

### 4.1 Сегменты пользователей

#### Сегмент A: Жители города (B2C)

```
Размер: ~60,000 человек
Конверсия в MAU (цель): 30% = 18,000 пользователей
```

**Подсегменты:**

| Подсегмент | Возраст | Потребности | % от сегмента |
|------------|---------|-------------|---------------|
| Молодежь | 18-25 | Развлечения, скидки, сообщества | 25% |
| Молодые семьи | 25-35 | Детские события, семейный досуг | 30% |
| Средний возраст | 35-50 | Бизнес-события, акции, услуги | 30% |
| Старшее поколение | 50+ | Культурные события, простой интерфейс | 15% |

#### Сегмент B: Местный бизнес (B2B)

```
Размер: ~2,500 бизнесов
Конверсия в активных (цель): 20% = 500 бизнесов
```

**Категории бизнеса:**

| Категория | % рынка | Потребности |
|-----------|---------|-------------|
| Рестораны/Кафе | 25% | Продвижение, акции, доставка |
| Услуги (салоны, ремонт) | 30% | Каталог, отзывы, бронирование |
| Торговля | 20% | Акции, каталог товаров |
| Спорт/Развлечения | 10% | Афиша, абонементы |
| Образование | 10% | Курсы, мастер-классы |
| Другое | 5% | Различные потребности |

### 4.2 Персоны пользователей

#### Персона 1: Алия (Молодая мама)

```
Возраст: 32 года
Семья: Замужем, 2 детей (4 и 7 лет)
Доход: Средний
Устройства: iPhone, активный пользователь Instagram
```

**Jobs to be Done:**
- Найти детские мероприятия на выходные
- Сэкономить на семейном досуге (скидки)
- Найти проверенные детские центры
- Общаться с другими мамами

**Пользовательский путь:**
1. Открывает приложение в пятницу вечером
2. Фильтрует по "Детские" события
3. Смотрит скидки в разделе "Дети"
4. Сохраняет понравившееся в избранное
5. Присоединяется к сообществу "Мамы Жезказгана"

---

#### Персона 2: Марат (Владелец кафе)

```
Возраст: 38 лет
Бизнес: Кафе "Арома", 5 сотрудников
Опыт в digital: Базовый (Instagram, WhatsApp)
Цель: Увеличить поток клиентов
```

**Jobs to be Done:**
- Привлечь новых клиентов
- Сообщить об акциях без затрат на рекламу
- Найти подрядчиков (фотограф, дизайнер)
- Организовать мероприятие в кафе

**Пользовательский путь:**
1. Регистрирует бизнес-аккаунт
2. Заполняет профиль кафе
3. Публикует акцию "1+1 на кофе"
4. Создает событие "Вечер настольных игр"
5. Ищет фотографа в коллаборациях

---

#### Персона 3: Данияр (Студент)

```
Возраст: 21 год
Статус: Студент, подработка
Интересы: Спорт, музыка, IT
Устройства: Android, активный в Telegram
```

**Jobs to be Done:**
- Узнать о бесплатных событиях
- Найти скидки на развлечения
- Найти единомышленников
- Быть в курсе городских новостей

**Пользовательский путь:**
1. Фильтрует бесплатные события
2. Подписывается на уведомления о концертах
3. Вступает в "Спортивный Жезказган"
4. Собирает команду на турнир через сообщество

## 5. Value Proposition

### 5.1 Value Proposition Canvas

#### Для жителей (B2C)

| Работы клиента | Боли | Выгоды |
|----------------|------|--------|
| Найти куда пойти | Информация разбросана по чатам | Всё в одном месте |
| Сэкономить деньги | Не знаю об акциях | Агрегатор скидок |
| Найти проверенные места | Нет отзывов/рейтингов | Верифицированный бизнес |
| Общаться по интересам | Нет платформы | Городские сообщества |

**Уникальное предложение:**
> "Узнай первым о всех событиях и скидках Жезказгана — бесплатно"

#### Для бизнеса (B2B)

| Работы клиента | Боли | Выгоды |
|----------------|------|--------|
| Привлечь клиентов | Дорогая реклама | Бесплатное размещение |
| Рассказать об акциях | Низкий охват | Целевая аудитория |
| Найти партнёров | Нет нетворкинга | Раздел коллабораций |
| Организовать событие | Сложно продвигать | Афиша с охватом |

**Уникальное предложение:**
> "Бесплатно расскажи о своём бизнесе 40,000 жителям Жезказгана"

### 5.2 Конкурентное преимущество

```
+------------------+     +------------------+     +------------------+
|    СОБЫТИЯ       | +   |     БИЗНЕС       | +   |   СООБЩЕСТВО     |
|    (Афиша)       |     |    (Каталог)     |     |    (Groups)      |
+------------------+     +------------------+     +------------------+
                    \           |           /
                     \          |          /
                      v         v         v
              +--------------------------------+
              |    ЕДИНАЯ ГОРОДСКАЯ           |
              |       ПЛАТФОРМА               |
              |    JEZKAZGAN CONNECT          |
              +--------------------------------+
```

**Ключевые отличия:**
1. **Гиперлокальность** — 100% фокус на одном городе
2. **Комплексность** — не просто афиша, а экосистема
3. **B2B + B2C** — ценность для обеих сторон
4. **Community-driven** — пользователи создают контент

