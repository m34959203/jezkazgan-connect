# Идея проекта Jezkazgan Connect

---

## Одной строкой

> **Jezkazgan Connect** — это цифровая платформа, которая объединяет жителей Жезказгана с местными событиями, бизнесами и друг с другом.

---

## Проблема

### Что происходит сейчас в Жезказгане?

**Для жителей:**
- Информация о событиях разбросана по десяткам Telegram-каналов и Instagram-аккаунтов
- Нет единого места, где можно узнать "куда сходить на выходных"
- Скидки и акции от местных бизнесов теряются в информационном шуме
- Сложно найти проверенный бизнес или услугу
- Нет площадки для объединения по интересам

**Для бизнеса:**
- Дорогая и неэффективная реклама (баннеры, листовки)
- Сложно достучаться до целевой аудитории
- Нет инструментов для анализа эффективности
- Негде найти партнёров для сотрудничества

## Как это работает?

### Для жителя города

```
Утро понедельника. Алия открывает Jezkazgan Connect...

1. АФИША
   → Видит, что в субботу концерт в ДК Горняков
   → Фильтрует "детские мероприятия" — находит праздник для детей
   → Сохраняет в избранное, получит напоминание

2. СКИДКИ
   → Листает акции — "Второй кофе бесплатно" в кофейне рядом с работой
   → Получает QR-код, покажет на кассе

3. БИЗНЕСЫ
   → Ищет салон красоты с хорошими отзывами
   → Находит верифицированный салон, звонит и записывается

4. СООБЩЕСТВО
   → Вступает в группу "Мамы Жезказгана"
   → Узнаёт о совместных прогулках и детских кружках
```

### Для владельца бизнеса

```
Марат владеет кафе "Арома". Вот как он использует платформу...

1. ПРОФИЛЬ БИЗНЕСА
   → Создаёт страницу кафе с фото, меню, контактами
   → Проходит верификацию — получает галочку доверия
   → Клиенты находят его в каталоге

2. АКЦИИ
   → Публикует акцию "1+1 на капучино до 12:00"
   → 500 человек видят акцию в ленте
   → 50 приходят с QR-кодом — Марат видит конверсию

3. СОБЫТИЯ
   → Организует "Вечер настольных игр" в кафе
   → Событие попадает в афишу города
   → Собирает 30 участников без затрат на рекламу

4. КОЛЛАБОРАЦИИ
   → Публикует запрос "Ищу фотографа для съёмки меню"
   → Получает 5 откликов, выбирает лучшего

5. АНАЛИТИКА
   → Видит: 1200 просмотров профиля, 80 переходов в WhatsApp
   → Понимает, что работает, а что нет
```

## Почему это сработает?

### 1. Реальная проблема
- Жезказган — изолированный город (300 км до Караганды)
- Нет федеральных сервисов типа Афиша.ру или 2ГИС с событиями
- Информация действительно разрознена

### 2. Готовая аудитория
- 90,000 жителей, ~60,000 интернет-пользователей
- Высокое проникновение мессенджеров
- Активное локальное сообщество

### 3. Нет конкурентов
- Первая платформа такого типа для города
- Telegram-каналы не заменяют полноценный сервис
- 2ГИС не покрывает события и сообщества

### 4. Взаимовыгодная модель
- Жителям — бесплатно
- Бизнесу — бесплатный базовый тариф
- Платные функции для тех, кому нужно больше

## Резюме

**Jezkazgan Connect** решает реальную проблему информационной разрозненности в городе, создавая ценность для обеих сторон:

- **Жителям** — единое место для поиска событий, скидок и сообществ
- **Бизнесу** — бесплатный канал продвижения с аналитикой

Проект имеет:
- Готовый frontend (90%)
- Понятную бизнес-модель
- Отсутствие конкурентов
- Потенциал масштабирования
